    • Rules

    hover over

  • The cursor changes from a pointer to a hand pointer when hovering over.
  • The image quickly changes from a more model centric photo to a photo that highlights the product.
  • A banner appears on the bottom of the product image giving the user the option to "Quickshop."
  • Another small button, a circle with a heart outline, also appears where a user can add an item to their wishlist.

    • hover off

  • The cursor changes from a a hand pointer to a pointer when hovering off.
  • The image quickly changes from a a photo that highlights the product to a more model centric photo.
  • A "Quickshop" banner disapears from the bottom of the product image.
  • Another small button, a circle with a heart outline, also disapeard when hovering off
